# Taskify

Taskify is a full-stack Task Manager application built with:

- **Frontend:** React (Vite)  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  

## Features

- User authentication (Register, Login)  
- Add, update, delete tasks  
- Responsive UI  

## Project Structure

Taskify/
│
├── frontend/ # React frontend
└── backend/ # Node.js backend


## How to Run Locally

### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
npm install
npm start
